# outputs/figures/

## 目录说明

此目录用于存放**训练曲线、对比图等可视化结果**。

## 文件类型

- 训练损失曲线图
- 评估指标曲线图（PSNR、SSIM等）
- 不同实验的指标对比柱状图
- 模型对比可视化图

## 命名规范

建议命名格式：
- `pix2pix_loss_curve.png`
- `pix2pix_vs_cyclegan_comparison.png`
- `metrics_comparison_barplot.png`

## 生成方式

- 使用 `src/eval/visualize.py` 生成
- 或在notebook中直接生成并保存

## 用途

- 报告图表素材
- 结果分析可视化
- 实验对比展示
