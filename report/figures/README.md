# report/figures/

## 目录说明

此目录用于存放**报告用图表**。

## 文件类型

- 实验报告中的所有图表
- 高质量图片（建议PNG格式，300 DPI）
- 三联图、对比图、曲线图等

## 文件组织

建议按章节组织：
- `fig1_dataset_samples.png` - 数据集样例
- `fig2_model_architecture.png` - 模型架构图
- `fig3_training_curves.png` - 训练曲线
- `fig4_results_comparison.png` - 结果对比
- `fig5_failure_cases.png` - 失败案例分析

## 注意事项

- 所有图表需有清晰的标题和编号
- 确保图片质量足够用于报告打印
- 保持图表风格一致
