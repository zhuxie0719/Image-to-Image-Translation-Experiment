# data/raw/

## 目录说明

此目录用于存放**原始拼接图像**（保持只读）。

## 数据格式

- Cityscapes 格式拼接图像：每张图片左侧为真实街景（photo），右侧为语义标签（label）
- 数据集下载地址：https://gitcode.com/open-source-toolkit/04615
- **图像尺寸**：512×256（左右拼接，每部分256×256）
- **图像格式**：JPG，RGB 3通道

## 数据集划分

数据集已经预划分：
- **训练集 (train)**: 2975 张图像
- **验证集 (val)**: 500 张图像
- **测试集 (test)**: 无（需要从验证集中划分）

## 注意事项

- 此目录为只读，请勿直接修改原始数据
- 所有数据处理操作应在 `data/processed/` 目录下进行
- 关于数据划分的详细说明，请参考 `data/splits/DATA_SPLIT_INFO.md`
